Responsive CSS Cheat Sheet

Key Responsive CSS Concepts

Filters

CSS filters allow you to apply graphical effects like blur or color shifting to elements.

filter: blur(5px);
filter: grayscale(100%);
filter: sepia(100%);

Media Queries

Media queries allow you to apply CSS styles depending on device general type or characteristics.

@media screen and (max-width: 600px) {
  /* styles */
}

Responsive Design

Responsive design ensures web pages render well on a variety of devices and window or screen sizes.

flex-wrap: wrap;
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

Typography

Responsive typography scales font sizes based on viewport width for better readability.

font-size: clamp(1rem, 2.5vw, 1.5rem);
line-height: 1.4;

Flexbox

Flexbox is a layout model that allows elements to align and distribute space within a container.

display: flex;
justify-content: space-between;
flex-wrap: wrap;

CSS Grid

CSS Grid is a two-dimensional layout system for the web.

display: grid;
grid-template-columns: 1fr 1fr 1fr;
gap: 20px;
